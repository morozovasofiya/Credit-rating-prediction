# Credit-rating-prediction

Учебный проект по анализу данных кредитного скоринга и прогнозированию кредитного рейтинга клиентов на основе финансовых и поведенческих признаков.

## О проекте

В проекте исследуются факторы, влияющие на кредитный рейтинг пользователя, и строятся модели для его предсказания.  
Работа включает статистический анализ признаков, проверку гипотез, исследование силы связи переменных с целевой категорией и построение классификационных моделей.

## Что сделано

— исследован датасет кредитного скоринга;

— выполнен анализ распределений признаков;

— проверены гипотезы о зависимости признаков от `Credit_Score`;

— рассчитаны меры связи между переменными;

— проведен отбор наиболее значимых признаков;

— построены модели классификации для предсказания кредитного рейтинга;

— выполнено сравнение однофакторных и многофакторных моделей;

— проведена оценка качества моделей по `accuracy` и confusion matrix.
